## 2) Chốt phạm vi MVP vs Future

### MVP (bắt buộc)
- Kết nối trading account (qua passview / read‑only).
- Auto sync giao dịch (delay OK).
- Dashboard vốn.
- Trade history (danh sách).
- Basic analytics (P/L, win rate, drawdown).

### Post‑MVP (tương lai)
- AI insights.
- So sánh nhiều account / dashboard gộp.
- Export / tax report.
- Screenshot tự động (optional).

**Đầu ra**
- Danh sách feature MVP (khóa phạm vi) + backlog Post‑MVP.

**DoD**
- Feature MVP được “freeze” cho vòng build hiện tại (muốn đổi phải có approval).
